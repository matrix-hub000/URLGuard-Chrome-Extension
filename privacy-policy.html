URLGuard Privacy Policy

Last updated: December 2024

Overview

URLGuard is a browser extension that helps protect users from phishing attacks and malicious websites by analyzing URLs in real-time. This privacy policy explains how we handle your data.

Data Collection and Usage

URLs Analyzed

URLGuard analyzes URLs you visit to determine if they are safe or potentially malicious. This analysis is performed locally and through third-party AI services when you have configured API keys.

API Keys

If you choose to configure API keys for enhanced analysis:

  • DeepSeek API Key: Used for AI-powered URL analysis. Your API key is stored locally in your browser and never shared with us.
  • OpenAI API Key: Used for online search functionality. Your API key is stored locally in your browser and never shared with us.

Local Storage

The following data is stored locally on your device:

  • Your API keys (if configured)
  • Analysis history (if enabled in settings)
  • Whitelisted and blacklisted domains
  • Extension settings and preferences

Data Sharing

URLGuard does not collect, store, or share any personal data with third parties. All data remains on your local device unless you explicitly configure API keys for enhanced analysis.

Third-Party Services

When you configure API keys, the extension may send URLs to:

  • DeepSeek API: For AI-powered URL analysis
  • OpenAI API: For online search functionality

These services are used only when you have provided your own API keys. We do not have access to your API keys or the data sent to these services.

Permissions

URLGuard requests the following permissions:

  • tabs: To access the current tab's URL for analysis
  • storage: To save your settings and preferences locally
  • webNavigation: To monitor URL changes for real-time protection
  • notifications: To alert you about potential threats
  • host_permissions: To inject warning banners on web pages

Data Security

All data is stored locally on your device using Chrome's secure storage APIs. Your API keys are encrypted and stored securely in your browser's local storage.

Your Rights

You have the right to:

  • Access your stored data through the extension settings
  • Delete your data by clearing the extension's storage
  • Disable data collection by turning off history storage
  • Remove API keys at any time

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be posted on this page with an updated revision date.

Contact

If you have any questions about this privacy policy, please contact us through the Chrome Web Store listing.
